
# `nikita.lxd.network.delete`

Delete an existing lxd network.

## Options

* `network` (required, string)   
  The network name.

## Callback parameters

* `err`   
  Error object if any.
* `status`   
  True if the network was deleted.

## Example

```js
require('nikita')
.lxd.network.delete({
  network: 'network0'
}, function(err, {status}){
  console.info( err ? err.message : 'Network deleted: ' + status);
})
```
